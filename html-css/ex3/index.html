Trabalhando carga de imagens

ex1:

<img src="logo-html.png" alt="Logo CSS">

  • tag img não possui fechamento: <img >
  • atributo src="": (Serve para colocar o caminho do arquivo)
  • atributo alt="": serve para colocar descrição da imagem (OBS: caso o navegador não encontre a imagem ir aparecer o conteúdo do alt. )

Abaixo você irá ver uma imagem que está dentro da mesma pasta, note que só precisamos utilizar o nome do arquivo seguido da estensão "natureza-html.jpg"

Podemos carregar imagens que estão dentro de uma pasta com o código abaixo.

<img src="fotos/logo-css.png" alt="Logo CSS">

Também podemos carregar fotos externas, e para isso devemos ter sua url completa como no exemplo abaixo.

<img src="https://www.casamidia.com.br/wp-content/uploads/2016/03/js-logo.png" alt="logo javascript">
